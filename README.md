# Install 

* [npm](https://nodejs.org/en/download/) 
* [hexo](https://hexo.io/docs/index.html)


```bash  
$ npm isntall hexo -g
$ npm install
```

# Colne custom theme

```bash  
$ git clone git@github.com:jianle/hexo-theme-corporate.git themes/corporate
$ cd themes/corporate
$ npm install
```

# Run server

```bash  
$ hexo s -g
```

# Deploy

```bash  
$./deploy.sh or hexo d -g
```
